# calendar


Instruction:
This calendar is used for timing user's events at work;
Also, the list of names is provided.

In this timetable the user can see all meetings schedledfor the meeting room; 
Filtration of all meeting by users' names is avaliable. 





To add new event press the link on the top of the table "Add event":
/*this will take user to the new page in your browser*/
	type the name of meeting
	select your name (and your partner's) //multiple choice
	select the day
	select time of event
	press the blue button "Add event" to submit

The form will not be submited if you selected the time and the day, where there is already a planned event.
The iformational bar will appear on your screen.

After a successful addition of an event the user can update the page with a schedule and see new event created.


To delete an event press the button with "X";
